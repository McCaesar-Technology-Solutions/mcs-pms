# MOJO APARTMENTS - User Guide

## Getting Started

Welcome to MOJO APARTMENTS - your complete property management solution for Ghana hospitality businesses. This guide walks you through using each feature.

## Navigating the Interface

### Topbar
The top navigation bar shows:
- **MOJO APARTMENTS logo** - Click to return to dashboard
- **Search** - Find guests, bookings, or rooms
- **Notifications** - Alerts and messages
- **Profile** - Your account settings

### Sidebar
Left navigation with links to:
- **Dashboard** - Overview and KPIs
- **Housekeeping** - Task management
- **Reservations** - Booking calendar
- **Guests** - Guest directory
- **Billing** - Invoices and payments
- **Channels** - OTA integrations
- **Analytics** - Performance metrics
- **GRA Reports** - Tax compliance
- **Bookings** - Booking management
- **Settings** - Configuration

## Dashboard

### Overview Section

The dashboard provides real-time insights:

**Key Metrics (4 cards)**
- **Total Revenue** - Combined revenue from all bookings this period
- **Occupancy Rate** - Percentage of rooms occupied (target: 75%+)
- **Avg Nightly Rate** - Average price per room per night
- **Total Bookings** - Number of active and upcoming reservations

### Room Availability

The 14-day strip shows room status:
- **Teal (occupied)** - Guest currently staying
- **Light teal (reserved)** - Booking confirmed but not yet checked in
- **Red (maintenance)** - Room unavailable for cleaning or repairs
- **Gray (available)** - Room open for new bookings

Click any day to see detailed room status.

### Upcoming Bookings

View next 5 reservations with:
- Guest name and confirmation status
- Room number
- Check-in and check-out dates
- Total price and nights stayed
- Booking source (Website, Airbnb, Booking.com, Walk-in)

Click "View All" to see complete reservations list.

### Task Summary

Housekeeping workload at a glance:
- **To Do** - Tasks awaiting assignment or starting
- **In Progress** - Currently being worked on
- **Done** - Completed tasks

Click task count to view specific tasks.

### Channel Performance

Revenue breakdown by booking source:
- Website
- Airbnb
- Booking.com
- Walk-in
- Other channels

Shows booking count and average rating for each channel.

### GRA Tax Compliance

Ghana Revenue Authority tracking:
- **Period** - Current tax filing month
- **Total Revenue** - Revenue subject to GRA reporting
- **Tax Amount** - Calculated GRA tax liability
- **Tax Rate** - Current applicable rate (typically 12%)
- **Invoices** - Number issued vs. paid

Status indicator shows filing status (Pending/Submitted/Approved).

### Staff Availability

See all housekeeping staff with:

## Reservations

- **Source** - Booking channel

#### Sorting & Filtering

Click column header to sort by:
- Guest name (A-Z)

- Status

Use filters to show:
- All bookings

- Only checked in
- By date range
- By source

### Booking Details

Click any reservation to view/edit:
- Guest contact information
- Room assignment
- Check-in and check-out times
- Number of guests
- Special requests
- Payment method and status
- Housekeeping notes
- Communication history

Make changes and save:
- Guest details

- Special requests
- Communication notes

Print invoice from details view.

### Guest Actions

From profile, you can:

## Billing

### Financial Overview

KPI cards showing:
- **Monthly Revenue** - Total bookings this month
- **Invoiced Amount** - Invoices sent to guests
- **Paid Amount** - Payments received
- **Outstanding** - Amount still owed
- **Collection Rate** - Percentage of revenue collected

### Invoice Management

View and manage invoices:
- **Date issued**
- **Guest name**
- **Invoice number**
- **Amount**
- **Status** - Draft/Sent/Paid/Overdue
- **Payment method**

#### Creating Invoices

1. Click "New Invoice"
2. Select guest or create new
3. Add line items (room charge, taxes, fees)
4. Set payment terms (Due on checkout, 7 days, 30 days)
5. Add payment methods
6. Send to guest
7. Guest receives via email with payment link

#### Recording Payments

1. Find invoice
2. Click "Record Payment"
3. Select payment method (Card, Bank Transfer, Cash)
4. Enter amount and date
5. Add reference (check number, transaction ID)
6. Save

Automatically updates collection metrics.

### Payment Methods

Configure accepted payment methods:
- Credit/Debit cards
- Bank transfers
- Mobile money (MTN, Vodafone)
- Cash on checkout
- Stripe/Square integration

## Channels

### OTA Integration Status

See connection status for each channel:
- **Website** - Direct bookings through your site
- **Airbnb** - Connected and syncing
- **Booking.com** - Connected and syncing
- **Expedia** - Connection pending setup
- **Google Hotels** - Integration available

### Connecting Channels

Click channel to connect:

1. **Airbnb**
   - Click "Connect Airbnb"
   - Authorize with Airbnb account
   - Select property to link
   - Sync automatically begins

2. **Booking.com**
   - Enter Booking.com property ID
   - Paste extranet username/password
   - Verify connection
   - Confirm booking sync

3. **Website**
   - Install booking widget
   - Configure checkout flow
   - Set commission rate
   - Enable payment processing

### Inventory Sync

Sync your room inventory across channels:
- **Auto-sync** - Real-time inventory updates
- **Manual sync** - Click "Sync Now" button
- **Sync schedule** - Run every 2 hours (configurable)
- **Sync status** - See last update time

### Booking Sync

Incoming bookings from channels automatically:
- Create reservation in system
- Assign room
- Alert housekeeping
- Send welcome email to guest
- Update channel with confirmation

## Common Workflows

### Daily Morning

1. Open Dashboard
2. Check occupancy for today
3. Review housekeeping tasks
4. Confirm check-in guests
5. Verify channel bookings synced
6. Alert team of any issues

### Guest Check-In

1. Go to Reservations
2. Find guest booking
3. Click "Check In"
4. Confirm guest details
5. Update any special requests
6. Provide room key and welcome packet
7. Guest checked in

### Processing Payment

1. Go to Billing
2. Find invoice
3. Click "Record Payment"
4. Enter amount and method
5. Confirm receipt
6. Invoice marked paid

### Month-End

1. Go to Analytics
2. Review monthly revenue
3. Go to GRA Reports
4. Generate tax report
5. Prepare for GRA submission
6. Download and send to accountant

## Tips & Best Practices

- **Update guest info** - Keep accurate data for communication
- **Use special requests** - Alert housekeeping to guest needs
- **Write detailed notes** - Help team understand issues
- **Monitor occupancy** - Adjust pricing based on demand
- **Sync channels daily** - Stay updated across all platforms
- **Track payments** - Know your cash flow status
- **Regular backups** - Export reports monthly
- **Team training** - Ensure all staff know the system
